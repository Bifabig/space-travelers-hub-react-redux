**Space Travelers' Hub** is a web application that uses real live data from the SpaceX API. It provides information on commercial and scientific space travel services. The application also allows users to book rockets and join selected space missions.

## ğŸ’» Getting Started <a name="getting-started"></a>

<!-- > Describe how a new developer could make use of your project. -->

### Prerequisites

To get a local copy up and running, follow these steps.

In order to run this project you need:

- A text editor like Microsoft Visual Studio Code.
- A Github account.
- Internet Connection.
- Basic HTML, CSS, JavaScript and React knowledge.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/Bifabig/space-travelers-hub-react-redux.git
```

### Install

Run this command inside the cloned repository

```sh
npm install
```

### Usage

To run the project, execute the following command:

```sh
npm start
```

### Run tests

To run tests, run the following command:

For unit testing

```sh
  npm test
```

For styling

```sh
  npx stylelint "**/*.{css,scss}"
```

For JavaScript

```sh
  npx eslint "**/*.{js,jsx}"
```
